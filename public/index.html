Why It's Built This Way

Real-time sync

A WebSocket hub pushes dashboard changes to every overlay in under 50ms. On this demo, a client-side shim stands in for the server.

Zero heavy deps

No React, no Tailwind, no build step. Just Express, ws, and vanilla front-end code that OBS Browser Sources love.

Synthesized audio

Alert sounds are generated on the fly with oscillators and envelopes - no bundled mp3s to load.

OBS-first sizing

Overlays are pixel-precise for 1920×1080 (and 1080×1920 vertical), so they drop straight into a scene.
